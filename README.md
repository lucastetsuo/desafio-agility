# Desafio Agility

Desafio proporcionado pela Agility no processo de admissão

### Pré-requisitos
 - PHP 7.2.5
 - Composer
 - Node

### Instalação
 - Clonar projeto
 - Executar comando "composer install"
 - Executar comando "npm install"
 - Executar comando "php artisan serve"
 - Acessar pela url "http://localhost:8000"
